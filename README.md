# PosomePa

> India's First Hourly Space Marketplace — Your Space, Your Time.

Discover and book workspaces, creative studios & event venues by the hour — no deposits, no lock-ins, zero friction.

[![Live Site](https://img.shields.io/badge/Live-posomepa.pages.dev-purple?style=flat&logo=cloudflare)](https://posomepa-web.pages.dev/)

## What is PosomePa?

PosomePa is a marketplace that lets you rent spaces — from coworking desks to photo studios, event venues to rooftop cafés — by the hour. No long-term commitments, no security deposits, no hassle.

## Features

| Feature | Description |
|---------|-------------|
| **₹0 Deposits** | Book instantly without any security deposit |
| **<30s Booking** | From open to confirmed in under 30 seconds |
| **AI Search** | Smart recommendations powered by AI |
| **KYC Verified Hosts** | All hosts verified for safety |
| **Instant Confirmation** | Get GPS directions & direct chat with host |
| **Free Cancellation** | Flexible cancellation policy |

## How It Works

1. **Search** — Browse spaces near you
2. **Book** — Pick your date, time & duration
3. **Go** — Pay via Razorpay & enjoy

## Tech Stack

- **Frontend**: Vanilla HTML/CSS/JS
- **Fonts**: Syne, DM Sans
- **Hosting**: Cloudflare Pages
- **CDN**: jsDelivr

## Development

```bash
# Clone the repo
git clone https://github.com/priiikrrrr/posomepa-web.git
cd posomepa-web

# Run locally (any static server)
npx serve .
# or
python -m http.server 8000
```
